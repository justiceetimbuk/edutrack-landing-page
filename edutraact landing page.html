Core Features

Everything Your School Needs
for Academic Excellence

From daily score entry to end-of-term report cards, EduTrack covers every step of the academic records process.

Smart Score Entry

Enter Tests, Assignments, Quizzes, Debates, Projects, Affective Domain, and Exam scores in one structured grid. CA totals and grades are calculated automatically.

Report Cards

Generate official, print-ready student report cards per term. Each card includes all subject scores, grades, remarks, and class position.

Broadsheet View

View all students × all subjects on one comprehensive sheet. Filter by class and term, then print with a single click.

Official Transcripts

Generate detailed academic transcripts for current students and alumni — printable and complete with full subject histories.

Alumni Records

Permanently store records for graduated students. Add historical scores and generate transcripts years after graduation.

Student Promotion

Promote entire classes to the next level at the start of each session in one action. Handles JSS and SSS progression automatically.

Teacher Management

Register teachers with departments, form classes, and subjects. Each teacher gets secure Firebase login credentials.

Subject Allocation

Assign subjects by class type — Science, Art, Commercial streams. Manage exactly which subjects appear for each class level.

Firebase Authentication

Enterprise-grade Google Firebase security. Role-based access for Admins, Teachers, and Students — with password reset built in.

Frequently Asked Questions

Got Questions? We’ve Got Answers

Everything you need to know before getting started with EduTrack.

How does EduTrack store school data — is it safe?
+
All data is stored on Google Firebase — the same infrastructure used by major global apps. Access is protected by role-based Firebase Authentication, so only authorised admins, teachers, and students can log in and see their permitted data.
Can teachers enter scores from their mobile phones?
+
Yes — 100%. EduTrack is fully mobile-first. Every feature, including score entry, works perfectly on any smartphone. Teachers do not need a laptop or computer to use the platform.
Can I use EduTrack for both JSS and SSS classes?
+
Absolutely. EduTrack supports all 12+ class levels — from JSS 1 through SSS 3, including Science, Art and Commercial streams. Subject allocation is customizable per class type, so each stream sees only its relevant subjects.
What happens if I enter a wrong score by mistake?
+
No problem. Any saved score record can be edited individually from the Saved Score Records table. Admins and teachers can click the edit button on any record to update individual CA components or exam scores at any time.
How many teachers can be registered on one account?
+
The Starter plan supports up to 5 teacher accounts. The School plan supports unlimited teachers. Each teacher gets their own secure login and can only access score entry for their assigned subjects and classes.
Can I generate transcripts for already-graduated students?
+
Yes. EduTrack has a dedicated Alumni Records module where you can store and retrieve records for graduated students indefinitely. Transcripts can be generated and printed at any time, even years after graduation.
Is there a free trial available?
+
Yes — you can request a free trial by contacting us on WhatsApp. We'll set up a demo account for your school so you can explore all features before making any payment commitment.
Does the school name and logo appear on report cards?
+
Yes. In the Settings panel, the admin can configure the school name, motto, address and principal's name. All of this information appears on generated report cards, broadsheets and transcripts — giving every document a professional, branded appearance.
